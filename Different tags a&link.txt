<link> => เป็น empty Element(tags <br>ไม่ต้องมี tag ปิด) จะไม่มีอะไรภายใน tags ใช้ระบุความสัมพันธ์กับ document อื่น และแทรัพยากรณ์ภายนอก จะอยู๋ในส่วน <head>
<a> => ในทางตรงกันข้าม tags ไม่ใช่empty Element(องค์ประกอบที่ว่างเปล่า) และระบุ Object ที่จะสร้างบนหน้า - เช่นลิงค์หรือรูปภาพที่คลิกได้  ซึ่งนำผู้ใช้ไปยังตำแหน่งอื่น แท็กนี้ใช้ใน<body>ส่วนเท่านั้น
